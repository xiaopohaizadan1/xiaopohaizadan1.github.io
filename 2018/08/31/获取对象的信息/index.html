获取对象的信息

python python 2018/08/31 阅读数: 1000

获取对象的信息

  1. type()

    判断变量的类型

    isinstance() 判断对象是否属于某个类型

  2. dir(‘abc’)

    可以获取对象的所有的属性和方法

    dir(‘ABC’)
    [‘__add__‘, ‘__class‘, ‘contains‘, ‘delattr‘, ‘dir‘, ‘doc‘, ‘eq‘, ‘format‘, ‘ge‘, ‘getattribute‘, ‘getitem‘, ‘getnewargs‘, ‘gt‘, ‘hash‘, ‘init‘, ‘init_subclass‘, ‘iter‘, ‘le‘, ‘len‘, ‘lt‘, ‘mod‘, ‘mul‘, ‘ne‘, ‘new‘, ‘reduce‘, ‘reduce_ex‘, ‘repr‘, ‘rmod‘, ‘rmul‘, ‘setattr‘, ‘sizeof‘, ‘str‘, ‘subclasshook__’, ‘capitalize’, ‘casefold’, ‘center’, ‘count’, ‘encode’, ‘endswith’, ‘expandtabs’, ‘find’, ‘format’, ‘format_map’, ‘index’, ‘isalnum’, ‘isalpha’, ‘isascii’, ‘isdecimal’, ‘isdigit’, ‘isidentifier’, ‘islower’, ‘isnumeric’, ‘isprintable’, ‘isspace’, ‘istitle’, ‘isupper’, ‘join’, ‘ljust’, ‘lower’, ‘lstrip’, ‘maketrans’, ‘partition’, ‘replace’, ‘rfind’, ‘rindex’, ‘rjust’, ‘rpartition’, ‘rsplit’, ‘rstrip’, ‘split’, ‘splitlines’, ‘startswith’, ‘strip’, ‘swapcase’, ‘title’, ‘translate’, ‘upper’, ‘zfill’]

    带有__的都是可以调用的方法

  3. Python 里面自带的len()函数，实际是调用的是对象的__len__()方法

    如果想要自己的写的类中有len方法可以自定义

  4. getattr() setattr() hasattr()

  5. hasattr(obj ,’x’ ,404) 对象中是否有x属性没有的话返回 404

  6. setattr(obj,’y’,19) 设置y属性的值为19

实例属性和类属性

实例属性

class Student(object):

def \_\_init__(self,name):

    this.name = name

obj = Student(“zhangsan”)

obj.name

obj.age = 123

类属性

class Student(object):

name = "STUDENT"

obj = Student()

obj.name 输出的 STUDENT

Student.name 输出的 STUDENT

setattr(obj,name,’李四’) 这样写会报错提示name没有定义

obj.name = ‘李四’

obj.name 输出的 李四

Student.name 输出的 STUDENT

del obj.name

obj.name 输出的 STUDENT

  • 实例属性属于各个不同的实例 互不干扰
  • 类属性是共有的
  • 不要将实例属性和类属性取同名的，不然会产生难以发现的错误

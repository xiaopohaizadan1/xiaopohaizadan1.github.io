访问限制

不能让外部的调用随意的修改属性 ，只能通过类中定义的方法来修改

需要将属性变为私有的 __ 在属性前面加上，新写方法来get和set属性

通过实例直接调用私有的属性会报错

class Student(object):

def \_\_init__(self, name.age):

    self.__name = name

    self.__age = age

def get_name(self):

    return self.__name

def set_name(self,name):

    self.__name = name

bart = Student(“xiaobai”,12)

bart.__name 报错，报没有这个属性

bart.get() 返回 xiaobai

bart.__name = ‘xiaohei’

bart.__name 返回 xiaohei 这个其实不是访问的私有变量，而是新建了一个属性

bart._Student__name仍然可以再class的外部 访问 (python 的解释器将私有的变量改成_Student__ name )

1.切片(Slice)

  1. d = [1,2,3]

    d[0:2] 返回 [1,2]

    d[0:1] 返回 [1]

  2. d[:3] 返回 [1,2,3] :前面的0可以省略

  3. d[-1] 是取倒数第一个元素

  4. d[:10:2] 前十个数，每两个取一个值

  5. d[::5] 所有数没五个取一个

  6. f = d[:] 复制

  7. tuple 的切片

    (1,2,3,4)[0:2] 返回 (1,2)

2.迭代

  1. for 。。。 in

    无论是list还是tuple都可以使用 for 。。。 in 来实现循环

  2. dict默认使用key来迭代的，如果想要值迭代的需要 dict.vlaues()

    for x in d:

    ​ print(x) 对key迭代

    for x in d.values():

    ​ print(x). 对值迭代

    for x in d.items():

    ​ print(x) 对key-value 迭代

  1. 判断一个对象是否是可以迭代的

    ​ import Iterable from collections

    ​ isinstance(‘abc’,Iterable) True str 是可以迭代的

    ​ isinstance([1,2,3],Iterable) True list 是可以迭代的

    ​ isinstance(123,Iterable) False 整数 是不可以迭代的

  2. enumerate() 将list变为索引-值的dict

3.列表生成式

  1. [x * x for x in range(1,11)]

    生成 一个 1-10 的平方的列表

  2. [x * x for x in range(1,11) if x%2 == 0] 偶数的平方

  3. 两层循环 生成全排列

    [m+n for m in ‘ABC’ for n in ‘XYZ’]

  4. import os

    [x for x in listdir(‘.’)] 列出当前目录下所有的文件

5.迭代器

  1. 可以作用于for 循环的对象都是Iterable对象

    isinstance()判断一个对象是否是Iterable对象

    form collections import Iterable

    isinstance([],Iterable)

    isinstance({},Iterable)

  2. 可以被next()函数调用的并且不断返回值的对象 称为Iterator迭代器

    form collections import Iterator

    isinstance([],Iterator) False

    instance((x for x in range(10)),Iterator) Ture

  1. 递归函数

    在函数中调用自身的函数

    def fact(n):

    ​ if n == 1 :

    ​ return 1

    ​ return n * fact(n - 1)

    栈溢出 函数调用是通过栈stack 数据结构实现的 每当进入一个函数 调用 栈会加一层栈帧 每当函数 返回 栈会减少一层栈帧 。 栈的大小不是无效的。当函数调用过多的时候，会导致栈溢出

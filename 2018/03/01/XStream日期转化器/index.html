XStresam日期转化器

后端 技巧 2018/03/01

1. 属性转化器

实现 SingleValueConverter 接口（String 与 Object 之间的转换）

XStream是有一个默认的DateConverter的转化器的，但是并不是所有的时间格式都能支持，所以才需要自定义转化器

接口有三个方法:

1.canConvert(Class aClass)

判断是否使用这个转化器

2.fromString(String s)

String 转成 任意类型 这边是转成Date

3.toString(Object o)

Object 转成 String

使用:

只需要在相应的属性上面加上@XStreamConverter(value = DateConverter.class)

1.机器学习的术语

a.标签

标签可以是我们要预测的东西,比如我们要判断一封邮件是不是垃圾邮件,垃圾邮件和非垃圾邮件就是标签。拿简单的线性回归方程来说的话,也就是常说的Y变量(因变量)。

b.特征

特征就是输入的变量(自变量),也就是线性回归方程中的X变量。

c.样本

* 样本
    * 有标签样本
    * 无标签样本。
这个其实也很好理解。假设 y = 2x + 1
(0,1),(1,3)等等这些就是有标签样本，具有明确的特征和标签

(1,?) 只含有特性，但是没有标签的就是无标签样本

使用的是有标签样本来训练得到一个模型，我们可以根据训练得到的模型来预测无标签样本的一个标签值是多少。

d.模型

模型定义了特征与标签之间的关系，可以理解为一个表达式
 * 模型有两个生命周期
    * 训练表示创建或学习模型。也就是说，您向模型展示有标签样本，让模型逐渐学习特征与标签之间的关系。
    * 推断表示将训练后的模型应用于无标签样本。也就是说，您使用训练后的模型来做出有用的预测 (y')。

2.线性回归 训练和损失

a.线性回归

如何求蝉鸣和温度的关系？

一般都是找一条线,让更多的点能够在这条线上,然后求这个线的表示的函数

***
由此其实我们可以得到一个函数 : y= kx+b ，这个函数其实就是蝉鸣和温度关系的一个模型
在机器学习中通常表示为 : y' = b + w1x1 
y' 表示 我们预测的值
b  表示 偏差 与y轴的截距 
x1 表示 特征
w1 表示 特征的权重

在上图中可能只有一个特征，那就是温度，其实在实际中，可能不只是一个特征

y' = b + w1x1 + w2x2 + w3x3 这个就是一个三个特征的模型

b.训练和损失

经验风险最小化:机器学习算法通过检查多个样本并尝试找出可最大限度减少损失的模型


上图所示的就很清楚的可以看到左侧模型的损失比右侧的大，而这边的算法就是为了找出一个损失的最小的模型。

怎么来判断损失最小呢？为此有创建了一个函数来计算损失

L2损失 (损失函数,平方损失) : (y - y')2  有标签样本的实际的标签值 和 预测标签值之间的平方差 

均方误差 MSE :每个样本的平均平方损失


* (x,y)指样本 
* prediction(x) 就是我们预测的模型 
* D 指的是样本的数据集
* N 是样本的数量

MSE只是众多损失函数中的一种。并不能适用所有的情形的下的损失计算。

c.降低损失

有了损失,我们所要做的就是降低损失，如何来降低损失呢？
<br/>
主要是迭代试错


通过样本中的特征输入获取到的预测的值 与 该样本的实际标签进行比较计算出损失。然后换模型迭代计算。

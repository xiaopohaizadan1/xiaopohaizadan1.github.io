如何在服务器上运行java项目

后端 java 2018/02/09 阅读数: 1000 
通过IDEA 写一个最基础的项目,这边就不做详细介绍了,如图:


项目的dao层使用的是jpa来操作数据库的。

配置信息如下:


一开始这边没有设置数据库连接池的信息，不知道是不是导致后面再次连接的时候报异常的原因,这边先记录下

项目写了一个最简单的一个save的方法，将数据存入的数据库中

项目完成之后使用maven打包，然后将本地生成的jar包上传到服务器上

然后执行

// 这个是运行jar包的    当前ssh窗口被锁定，可按CTRL + C打断程序运行，或直接关闭窗口，程序退出 
java -jar jar包名.jar

// 这个是让项目在后台可以一直的运行 &代表在后台运行。
nohup java -jar jar包名.jar & 

nohup 意思是不挂断运行命令,当账户退出或终端关闭时,程序仍然运行

当用 nohup 命令执行作业时，缺省情况下该作业的所有输出被重定向到nohup.out的文件中，除非另外指定了输出文件
